Module 9 - Basic knowledge of  TA Frameworks: Layers, Runner, Business Objects

# Requirements

+ Node version 8 or above
+ Java

# Installation
````
npm install
````
# Run tests
````
npm run test
````

# Options
* Run tests using firefox browser
```
npm run test -- --firefox
```

* Several browser instances
```
npm run test -- --instance <number>
```
